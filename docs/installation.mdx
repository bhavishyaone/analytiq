## Option A — npm (Most Common)

npm comes pre-installed with Node.js. This is the most common way to install packages.

```bash
npm install analytiq
```

## Option B — yarn

If your project uses yarn instead of npm:

```bash
yarn add analytiq
```

<Note>
  Don't have yarn? Install it with: `npm install -g yarn`
</Note>

## Option C — pnpm

If your project uses pnpm:

```bash
pnpm add analytiq
```

<Note>
  Don't have pnpm? Install it with: `npm install -g pnpm`
</Note>

## Option D — CDN (No npm/Node.js Required)

If you have a plain HTML website with no build process, use this script tag instead of installing via npm:

```html
<script type="module">
  import { init, track, identify, reset, batchTrack } from 'https://cdn.jsdelivr.net/npm/analytiq/dist/index.js'

  init('pk_live_YOUR_KEY_HERE')

  track('page_view', { path: window.location.pathname })
</script>
```

Paste this inside your HTML file, just before the closing `</body>` tag.

<Info>
  All four installation methods (npm, yarn, pnpm, CDN) install the exact same SDK with identical features. Use whichever method your project is already using.
</Info>

## Verifying the Installation

After installing, run your app:

```bash
npm run dev
```

Open your browser, press **F12** → **Console** tab. After adding `init()` to your code, you should see:

```
[analytiq] Initialized successfully.
```

If you see this message — your installation is complete!
