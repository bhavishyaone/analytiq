<Info>
  **Which method should I use?**

  | Your project | Method to use |
  |-------------|--------------|
  | Plain `.html` files, no build tools | **CDN** (Option A below) |
  | Uses Vite, Webpack, or Parcel (but no framework) | **npm** (Option B below) |
  | A `.html` file inside a React/Vue project | Use the React or Vue guide instead |
</Info>

## Option A — CDN (No npm, No Installation)

This is the simplest method. You paste one `<script>` tag into your HTML — nothing to install.

### Step 1 — Open your HTML file

Open `index.html` (or whichever `.html` page you want to track) in your code editor.

### Step 2 — Paste just before `</body>`

Find the closing `</body>` tag at the bottom of your HTML file. Paste this block just above it:

```html
<!-- Paste this just before </body> -->
<script type="module">
  import { init, track } from 'https://cdn.jsdelivr.net/npm/analytiq/dist/index.js'

  // Step 1: Start Analytiq with your API key
  init('pk_live_YOUR_KEY_HERE')

  // Step 2: Track that this page was viewed
  track('page_view', {
    page: document.title,
    path: window.location.pathname
  })
</script>
```

<Warning>
  Replace `pk_live_YOUR_KEY_HERE` with your actual API key from the Settings page.
</Warning>

Your full `index.html` will look like this:

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8" />
  <title>My Website</title>
</head>
<body>

  <h1>Welcome to My Website</h1>
  <button id="signupBtn">Sign Up Free</button>

  <!-- Analytiq — paste just before </body> -->
  <script type="module">
    import { init, track } from 'https://cdn.jsdelivr.net/npm/analytiq/dist/index.js'

    init('pk_live_YOUR_KEY_HERE')
    track('page_view', { page: document.title, path: window.location.pathname })
  </script>

</body>
</html>
```

### Step 3 — Track a button click

Give your button an `id` attribute, then listen for its click:

```html
<!-- Your existing button — add an id if it doesn't have one -->
<button id="signupBtn">Sign Up Free</button>

<script type="module">
  import { init, track } from 'https://cdn.jsdelivr.net/npm/analytiq/dist/index.js'

  init('pk_live_YOUR_KEY_HERE')

  // Track the button click
  document.getElementById('signupBtn').addEventListener('click', function() {
    track('signup_clicked', { button: 'hero_cta' })
  })
</script>
```

### Step 4 — Track across multiple pages

If your site has multiple HTML pages (e.g., `about.html`, `contact.html`), paste the init + track code in **each file**:

```html
<!-- Put this in every .html page you want to track -->
<script type="module">
  import { init, track } from 'https://cdn.jsdelivr.net/npm/analytiq/dist/index.js'

  init('pk_live_YOUR_KEY_HERE')

  // document.title and window.location.pathname automatically
  // change on each page — no extra code needed
  track('page_view', {
    page: document.title,
    path: window.location.pathname
  })
</script>
```

<Tip>
  To avoid repeating the code on every page, create a shared `analytics.js` file and import it with `<script type="module" src="analytics.js"></script>` on every page.
</Tip>

### Step 5 — Verify it's working

1. Open your `.html` file in a browser (double-click it, or run a local server)
2. Press **F12** → click the **Console** tab
3. You should see: `[analytiq] Initialized successfully.`
4. Go to your Analytiq dashboard → Events

## Option B — npm + Bundler (Vite, Webpack, Parcel)

Use this if you have a bundler set up but no framework like React or Vue.

### Step 1 — Install the SDK

```bash
npm install analytiq
```

### Step 2 — Create `analytics.js`

Create a new file called `analytics.js` (or `src/analytics.js`) and paste:

```js
import { init, track } from 'analytiq'

// Initialize Analytiq
init('pk_live_YOUR_KEY_HERE')

// Track the current page
track('page_view', {
  page: document.title,
  path: window.location.pathname
})
```

### Step 3 — Import this file in your main entry

Open your bundler's entry file (usually `main.js` or `index.js`) and import it:

```js
import './analytics.js'   // ← ADD this at the top

// ... rest of your existing code
```

### Step 4 — Track a button click (in any JS file)

In whichever JS file handles your button:

```js
import { track } from 'analytiq'   // ← ADD this import

// Add ONE line inside your existing click handler
document.getElementById('signupBtn').addEventListener('click', function() {
  track('signup_clicked', { page: 'home' })  // ← ADD this line
})
```

### Step 5 — Run and verify

```bash
npm run dev
```

Open your browser → **F12** → **Console** → look for `[analytiq] Initialized successfully.`

## Advanced — Send multiple events at once (both methods)

Instead of separate `track()` calls, send multiple events in one shot using `batchTrack()`:

<Tabs>
  <Tab title="CDN">
    ```html
    <script type="module">
      import { init, batchTrack } from 'https://cdn.jsdelivr.net/npm/analytiq/dist/index.js'

      init('pk_live_YOUR_KEY_HERE')

      batchTrack([
        { name: 'page_view', properties: { page: document.title } },
        { name: 'session_start' },
      ])
    </script>
    ```
  </Tab>
  <Tab title="npm + Bundler">
    ```js
    import { batchTrack } from 'analytiq'

    batchTrack([
      { name: 'page_view', properties: { path: window.location.pathname } },
      { name: 'session_start' },
    ])
    ```
  </Tab>
</Tabs>

## Identifying users (both methods)


If your HTML site has a login system, call `identify()` after login to link events to a specific user. Add this inside your existing login success handler:

```html
<script type="module">
  import { identify, reset } from 'https://cdn.jsdelivr.net/npm/analytiq/dist/index.js'

  // After login succeeds — paste inside your existing login function
  function onLoginSuccess(userId) {
    identify(userId)    // ← ADD this line
  }

  // After logout — paste inside your existing logout function
  function onLogout() {
    reset()             // ← ADD this line
  }
</script>
```
